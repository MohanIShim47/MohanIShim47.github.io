## 📖 Overview

MSPortal is a personal web platform designed to act as a hub for all my websites and web-based projects.
It simplifies access, improves organization, and provides a scalable foundation for future development.

## ✨ Features

- 🌐 Centralized Dashboard — Access all websites from a single interface
- 📁 Project Organization — Keep sites structured and easy to navigate
- ⚡ Scalable Architecture — Easily add new projects over time
- 🔧 Customizable — Built to evolve with my needs

## 🎯 Goals

- Create a clean and efficient portal for managing multiple sites
- Reduce friction when switching between projects
- Build a foundation for future automation and tooling

## 🚧 Status

This project is currently under active development.
Features and structure may change as it evolves.

- HTML/CSS/JS
- Jekyll
- Github Pages
