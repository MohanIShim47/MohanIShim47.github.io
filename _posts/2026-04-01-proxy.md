# What is a proxy?

A proxy is an *unblocker*, as most of you already know. In this post, I (TEDA) will go into detail about how proxies work, what they do, and how proxy sites are built using these ideas.

---

## How does a proxy work?

A proxy is like the “middleman” in a game of monkey in the middle.

Normally, when you visit a website, your browser sends a request directly to that site:
> “Hey, send me your content so I can display it.”

At the same time, the website can see your IP address (which identifies your network).

When you use a proxy, the process changes:

1. Your browser sends the request to the **proxy server** instead of the website  
2. The proxy forwards that request to the website  
3. The website responds to the proxy  
4. The proxy sends the data back to you  

Because of this, the website only sees the **proxy server's IP**, not yours.

---

## What do proxies do?

In simple terms, proxies help *hide* or *mask* your identity online.

In a school setting, this matters because:

- Schools often use filtering systems to block certain websites  
- These systems monitor traffic coming directly from your device/IP  
- A proxy reroutes your traffic, so the filter may only see a connection to the proxy, not the final destination  

This can allow access to sites that would normally be blocked.

---

## What is a proxy site?

A proxy site is a website that lets you use a proxy directly from your browser—no installation needed.

You typically:
1. Go to the proxy site  
2. Enter a URL (like YouTube or a game site)  
3. The proxy site loads that page *through its server*  

From your perspective, it feels like you're browsing normally—but everything is being routed through the proxy.

---

## How proxy sites actually work

Most proxy sites are built using backend code (like Node.js, Python, or PHP). Here's a simplified breakdown:

- The proxy server makes requests on behalf of the user  
- It fetches HTML, CSS, JavaScript, and other assets  
- It rewrites links so everything continues to pass through the proxy  
- It sends the modified content back to your browser  

Some advanced proxies also:
- Rewrite scripts
- Handle cookies
- Support WebSockets
- Try to avoid detection by filters

## Final thoughts

Proxies are powerful tools that act as a bridge between you and the internet. They can help bypass restrictions and add a layer of privacy, but they also come with limitations and risks.

Understanding how they work is the first step to using them effectively—and responsibly.
